A menu overlay activated by a floating action button. The pink button in the lower right corner of this page is a sample usage. No JavaScript coding is required for basic application of this component.

This component is framework-agnostic, so it can be used with any UI framework or even with just standard HTML.

Basic usage

1. Include zUIx library

Add the following line, preferably inside the head section of the HTML page:

<script src="//genielabs.github.io/zuix/js/zuix.min.js"></script>

this is the library that does all the magic : )

2. Add the menu markup

Put inside the field items the code for your menu items.


  

    
    Menu Item 1
    Menu Item 2
    Menu Item 3

... and that's it, super duper easy : )

Option attributes

  • data-o-scroller optional
    if the scrolling container it is not the main document, use this parameter to specify the value of data-ui-field attribute assigned to the container with the scroll.
  • data-o-button-color optional
    color of the menu button. The defaul value is deeppink.
  • data-o-icon-color optional
    color of the menu button icon. The defaul value is white.
  • data-ui-context optional
    identifier name to be used to reference this component from JavaScript.

Theming

You can customize the appearance of the menu button by overriding the template fields:

  • add menu_button for the normal state button
  • close menu_button_close for the open state button

For example the example menu shown in this page is using a Material Design Lite theme.

Scripting

Event listeners

var menuOverlay;
// since the component loads asynchronously
// a callback is required to ensure the component is ready
zuix.context('my-menu-id', function () {
    // add event listeners
    this.on('open', function() { /* ... */})
        .on('close', function() { /* ... */});
    // store a global reference of the
    // component for eventual later use
    menuOverlay = this;
});

Programmatically show/hide

// show the menu (enable)
menuOverlay.hide();
// hide the menu (disable)
menuOverlay.hide();
// show the menu button
menuOverlay.showButton();
// hide the menu button
menuOverlay.hideButton();

Also consider taking advantage of all other classes and utility methods included with zUIx library API which includes a DOM manipulation facility as well (zuix.$ - ZxQuery class).
